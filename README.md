# UED

このリポジトリは現在ゲーム（ビジュアルノベル風）用の静的ファイル群を含みます。

- 目的: プロトタイプのストーリー分岐とルートのプレースホルダを管理する。
- 主なファイル:
  - `index.html` — 中核のシングルページ（シナリオエンジン）
  - `route_logic.html`, `route_revenge.html`, `route_fear.html` — ルート用プレースホルダ
  - `climax.html` — 最終局面プレースホルダ

この README は簡易プレースホルダです。詳細なビルドや実行手順があれば追記してください。
